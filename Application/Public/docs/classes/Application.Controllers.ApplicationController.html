The ChickenWire controller class

This is the basis for all Controllers in your Application. To create a new controller simply extend this class. Any public function you define in your controller can then be used to route to.

Configurators

You can add one or more of the following configurators to your controller to add features.

requiresAuth

If your actions require a user to be logged in, you can define an Auth object in your configuration, and specify its name in your controllers. See Auth for more information.

static $requiresAuth = "BMK";            // Authentication required for all actions
static $requiresAuth = array("BMK",
    "except" => array("index", "show")      // No authentication required for index and show actions
);
static $requiresAuth = array("BMK",
    "only" => array("secret")               // Authentication only required for the secret action
);
